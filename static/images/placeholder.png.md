# Placeholder Afbeelding

Dit bestand dient als placeholder voor productafbeeldingen.

In een echte implementatie zou hier een PNG afbeelding staan.
Voor deze demo wordt een standaard icoon gebruikt als fallback.

Om een echte placeholder toe te voegen:
1. Vervang dit bestand door een PNG afbeelding
2. Noem het bestand `placeholder.png`
3. Gebruik een eenvoudig icoon of placeholder grafiek

Aanbevolen afmetingen: 400x400 pixels
